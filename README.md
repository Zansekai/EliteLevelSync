# EliteLevelSync

Bloque l'équipement d'items EliteMobs si le joueur n'a pas le niveau requis selon **mcMMO**, **ValhallaMMO** ou **MMOCore**.

---

## Prérequis

| Logiciel | Version minimale | Lien |
|----------|-----------------|------|
| Java JDK | 21 | https://adoptium.net |
| Maven | 3.8 | https://maven.apache.org/download.cgi |
| Paper/Spigot | 1.21.x | — |
| EliteMobs | 9.x | Obligatoire |
| mcMMO / ValhallaMMO / MMOCore | N'importe | Au moins un |
